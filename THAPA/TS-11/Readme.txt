MAP & FILTER
============

MAP 
===

Array map method takes a call back function which transforms the array by applying logics to each and every array element and returns a
new array.It does not mutate/change the original array.



FILTER
======

Array filter method takes a call back function which filters the array by applying logics to each and every array element and returns a
filtered array.It does not mutate/change the original array.
